Synthèse Inverse Audio (Stage IRIT)

Conception d'un prototype analysant un signal audio pour en déduire les paramètres de synthèse (oscillateurs, filtres, ADSR) et le reproduire.

Code sur GitHub

Aperçu du Projet

Réalisé dans le cadre de ma troisième année de Double Licence sous l'encadrement de Julie Mauclair au laboratoire IRIT, ce stage de recherche vise à explorer la synthèse inverse. L'objectif est de développer un modèle capable d'inférer les paramètres spécifiques d'un synthétiseur à partir d'un fichier audio source.

En mêlant intelligence artificielle (réseaux de neurones) et traitement du signal (analyse de Fourier, manipulation audio avec Python), ce projet confirme mon choix de spécialisation vers l'ingénierie audio et le traitement du signal.

Outils & Technologies

Python
Librairie PyO
Traitement du Signal (DSP)
Réseaux de neurones
Analyse de Fourier
